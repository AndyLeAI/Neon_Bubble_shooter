# Assets

## images
- `aurora-bg.png`: nền game.
- `launcher.png`: súng bắn bubble.
- `logo.png`: logo màn hình mở đầu.
- `bubbles/*.png`: 6 bubble sprite bóng sáng.

## animations
- `pop-spritesheet.png`: 10 frame nổ bubble.
- `sparkle-spritesheet.png`: 8 frame sparkle để mở rộng hiệu ứng sau này.

## audio
- `neon-orbit.wav`: nhạc nền loop.
- `shoot.wav`, `bounce.wav`, `pop.wav`, `drop.wav`, `tick.wav`, `win.wav`, `lose.wav`: hiệu ứng âm thanh gameplay.


## backgrounds/
- `bg-neon-01.png` đến `bg-neon-04.png`: các hình nền neon có thể thay trực tiếp trong game hoặc dùng làm asset dự phòng.

## fx/
- `glow-cyan.png`, `glow-pink.png`, `glow-violet.png`: glow elements để bạn dùng tiếp cho particle, UI hoặc effect.

## audio/extras
- `music-neon-dream.wav`: loop nền phụ.
- `music-cosmic-pulse.wav`: loop nền phụ.
- `swap.wav`: âm thanh đổi bóng.
- `combo.wav`: âm thanh combo.
- `ui-click.wav`: âm thanh bấm nút.


## images/powerups
- `bomb.png`: quà bom.
- `fire.png`: quà mồi lửa.
- `clock.png`: quà đồng hồ 15 giây.

## audio powerups
- `bomb.wav`: tiếng nổ bom.
- `fire.wav`: tiếng lửa quét hàng.
- `clock.wav`: tiếng kích hoạt đóng băng thời gian.
